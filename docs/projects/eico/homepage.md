## 项目介绍

公司品牌官网，涵盖案例展示、服务接洽、人才招聘等模块，响应式完美支持移动端。

## 主要成就

- 使用 Bootstrap 实现页面排版、Less 编写样式，支持响应式页面
- 使用 Backbone 实现组件化拆分、RequireJS 实现按需加载
- 使用 Swiper 实现轮播图、瀑布流布局 + Lazyload 展示部分页面
- 使用 Express(Node.js) 框架搭建 Http Server
